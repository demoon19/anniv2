# 🎉 Anniversary Web App

A simple and interactive anniversary web application designed to create a memorable and personalized experience. This project combines animations, visual elements, and user interaction to deliver a digital celebration page.

---

## 🚀 Overview

This project is a **web-based anniversary page** that can be used to:

- 💌 Deliver special messages  
- 🎨 Display aesthetic animations  
- ❤️ Create a personalized romantic or celebration experience  

Perfect for:
- Anniversary surprise 🎂  
- Birthday greeting 🎁  
- Personal web gift 💝  

---

## ✨ Features

- 🎬 Interactive animations
- 💖 Romantic / celebration theme UI
- 📱 Responsive design (mobile-friendly)
- 🎵 Background music support *(if available)*
- 🖱️ Click-based interactions / transitions
- 🎨 Clean and aesthetic design

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## 📂 Project Structure

```
anniv2-main/
│── index.html        # Main page
│── style.css         # Styling
│── script.js         # Interactivity
│── assets/           # Images, audio, etc
```

---

## ⚙️ Installation & Setup

1. Clone repository:
```bash
git clone https://github.com/username/anniversary-web.git
```

2. Masuk ke folder:
```bash
cd anniv2-main
```

3. Jalankan di browser:
```bash
open index.html
```

Atau gunakan **Live Server** di VS Code.

---

## ▶️ Usage

1. Buka website  
2. Nikmati animasi pembuka  
3. Klik atau interaksi sesuai instruksi  
4. Lihat pesan spesial yang ditampilkan
